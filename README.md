# OpenbreweryApp
 Учебный проект, выоплненный под контролем и проверкой кураторов на интенсиве ШИФТ от компании ЦФТ.
 Стек:
 - Jetpack compose
 - Hilt
 - Room
 - Ktor client
 - Kotlin flow
 - Kotin coroutines
